# Array

## Time Complexity
| Operation | Time Complexity |
| --------- | --------------- |
| Access    | $O(1)$            |
| Appending | $O(1)$            |
| Pop       | $O(1)$            |
| Search    | $O(n)$            |
| Insertion | $O(n)$            |
| Removal   | $O(n)$            |
| Sorting   | $O(n log n)$      |
| Slice     | $O(k)$            |
